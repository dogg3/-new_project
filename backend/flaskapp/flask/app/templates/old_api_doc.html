FootyPhi

An API serving statistically processed footballsdata from the 5 top football leauges in europe from - England, France, Germany, Spain and Italu. Defenders, midfielders and forwards have specific rankings but also general statistics easily available.The api is primarly serving eascouting.com with relevant rankings.
More information: https://helloreverb.com
